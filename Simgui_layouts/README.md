# SimGUI Layouts

Pre-configured layouts for the WPILib Simulation GUI. Each subfolder contains a `simgui.json` file and any associated assets (icons, images).

## How to use

1. Pick the layout you want from the subfolders below.
2. Copy the `simgui.json` file (and any images) into the **project root**.
3. Run the simulator — the GUI will load the copied layout automatically.

> **Note:** The root `simgui.json` is git-ignored, so your local copy won't be committed. These subfolder copies are the checked-in "templates."

## Available layouts

| Folder | Description |
|--------|-------------|
